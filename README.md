# Post Spring Best Practice Client

React, TypeScript, Vite, Tailwind CSS, Radix UI, TanStack Query, Vitest, and Playwright starter for product-facing frontend development.

## Requirements

- Node.js 24 LTS
- pnpm 10.x via Corepack

## Scripts

- `pnpm dev` starts the Vite dev server.
- `pnpm build` type-checks and builds the app.
- `pnpm validate` runs lint, type-check, and unit tests.
- `pnpm e2e` runs Playwright tests.
